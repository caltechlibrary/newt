Newt User Manual

What is in the Newt toolbox?

  • Newt, is a development tool. It’s primary task is to use intepret the YAML modeled data and render code, templates and configuration to run your app. The Newt command can also function as an app runner. As an app runner it is capabile of running Newt’s router and template engine and selected JSON data sources such as Postgres+PostgREST.
  • Newt’s router (i.e. ndr) is a web service that manages your requests and data pipelines as well as static file hosting
  • Newt’s template engine (i.e. nte) is a simple stateless template engine as a web service

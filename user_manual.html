Newt User Manual

What is in the Newt toolbox?

  • Newt, is a development tool. It provides a code generator as well as the ability to run NewtRouter, NewtMustache and PostgREST as you construct and test your data models for your application.
  • Newt Router is a web service that manages your requests and data pipelines as well as static file hosting
  • Newt Mustache is a web service providing a light weight template engine

About this software

Newt 0.0.7-dev

Authors

  • R. S. Doiel

Newt is an experimental web service providing data routing and a multistage pipeline. The pipeline can contain services that work with JSON data. Example object storage, form data validation, JSON transformation, even static file services. Newt can route data from a request to a JSON data source (e.g. Postgres+PostgREST or Solr) and then run those results through a pipeline that may include Newt’s Mustache template engine.

  • GitHub: <git+https://github.com/caltechlibrary/newt>
  • Issues: <git+https://github.com/caltechlibrary/newt/issues>

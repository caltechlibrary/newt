About this software

Newt 0.0.6

Authors

  • R. S. Doiel

Newt is an experimental microservice providing two stage data routing, form data validation, simple static file services. Newt can route data from a request to a JSON data source (e.g. Postgres+PostgREST) and then take the results and run them through Pandoc server.

  • GitHub: <git+https://github.com/caltechlibrary/newt>
  • Issues: <git+https://github.com/caltechlibrary/newt/issues>

Programming languages

  • YAML
  • SQL
  • Golang

Operating Systems

  • Linux
  • Windows
  • macOS

Software Requiremets

  • Postgres >= 15
  • PostgREST >= 12
  • Pandoc >= 3
  • Golang > 1.21
  • A front end web server supporting reverse proxy (e.g. Apache2, NginX)
